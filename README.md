# Python Practice Questions

Welcome to my Python practice repository! This project serves as a dedicated space for honing my programming skills through various challenges and exercises, covering everything from fundamental syntax to advanced language features.

## 🚀 Goals
- Master Python syntax and **PEP 8** best practices.
- Explore **Advanced Python** features like decorators, generators, and context managers.
- Solve real-world logic problems and automation tasks using clean, "Pythonic" code.

## 📁 Repository Structure
The exercises are organized by topic and complexity to make navigation easy:
- `/Basics`: Variables, loops, conditional logic, and fundamental data types.
- `/Advanced`: Decorators, generators, context managers, and multithreading.
- `/OOP`: Classes, inheritance, encapsulation, and polymorphism.
- `/Projects`: Small-scale automation scripts and logic-based applications.
